## Station emulator
*Work in progress*

Electric vehicle charging station emulator which supports OCPP 2.0.

### List of supported messages

- [ ] - BootNotification
- [ ] - SetVariables
- [ ] - ChangeAvailability
- [ ] - Heartbeat
- [ ] - StatusNotification
- [ ] - TransactionEvent
- [ ] - RequestStartTransaction (only remote authorization)
- [ ] - RequestStopTransaction
- [ ] - ReserveNow
- [ ] - CancelReservation
